Описание сайта

Этот сайт представляет собой программный комплекс, разработанный на языке Java, который обеспечивает возможность подготовки документов по шаблону в форматах .doc и .docx. Для достижения этой цели используется библиотека Apache POI.

Основные возможности:

  • Загрузка шаблонов документов в формате .doc и .docx
  • Автоматическое заполнение шаблонов данными
  • Поддержка сложных документов с таблицами, изображениями и другими элементами
  • Экспорт готовых документов в формате .doc и .docx

Основные технологии:

  • Java: Основной язык программирования, используемый для разработки серверной части приложения
  • Spring Boot: Фреймворк для создания микросервисов и веб-приложений на языке Java
  • Thymeleaf: Шаблонизатор для динамической генерации HTML-страниц на стороне сервера
  • Bootstrap: CSS-фреймворк для создания адаптивного дизайна
  • Apache POI: Библиотека для работы с документами в формате Microsoft Office
